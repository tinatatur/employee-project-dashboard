# Employee Project Dashboard

A simple HTML dashboard for tracking employee project assignments and status. This is a test/demo application used to verify Open Exchange platform features (listing page, README-as-description, categories, tags, etc.).

## What it does

- Displays a list of employees and their assigned projects
- Shows project status (e.g. In Progress / Completed / On Hold)
- Static front-end, single `index.html` file — no backend required

## Tech stack

HTML, CSS, JavaScript (client-side only)

## Run locally

Clone the repo and open `index.html` in a browser:

\`\`\`
git clone https://github.com/tinatatur/employee-project-dashboard.git
cd employee-project-dashboard
open index.html
\`\`\`

## Purpose

This repository is used as test data for QA of the InterSystems Open Exchange platform (test-app-oex listing) — not intended for production use.
